# Ficha Técnica de Filmes

Este repositório contém códigos desenvolvidos para fins de aprendizado de **Programação Orientada a Objetos (POO) em Java**.

## 📌 O que foi praticado
- Criação de uma classe (`Filme`) para representar um objeto do mundo real.
- Definição de **atributos**.
- Criação de **objetos da classe** e atribuição de valores aos atributos.
- Implementação de **métodos com e sem retorno**.
- Uso de **métodos de acesso (getters e setters)**.
- Aplicação de **encapsulamento**.
- Uso de `constructor` para inicialização de objetos.
- Uso do **operador `instanceof`** para verificação de tipo de objeto.
- Sobrescrita do método **`toString()`** para representação em texto do objeto.
- Utilização de **listas (`ArrayList`)** para armazenar múltiplos filmes.
- Uso de **Collections (`Collections.sort`)** para ordenação de dados.

## 🎯 Objetivo
Fixar os conceitos fundamentais de **classe, objeto, atributos e métodos em Java**, além de praticar a organização e manipulação de objetos dentro de uma aplicação simples.
